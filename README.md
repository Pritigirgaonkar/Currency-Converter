# 💱 Currency Converter

A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript. This project allows users to convert currencies in real-time using live exchange rates.

---

## 🚀 Features

- Real-time currency conversion
- Clean and user-friendly interface
- Responsive design
- Supports multiple currencies:
  - USD
  - EUR
  - INR
  - GBP
  - JPY
- Error handling for invalid input

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Exchange Rate API
